Terms of Service ini berlaku secara umum untuk:

## 2. Deskripsi Umum Software

Conot adalah software bot Discord untuk notifikasi konten YouTube yang dapat bekerja menggunakan:

- RSS feed resmi YouTube
- scraping ringan halaman channel atau hasil pencarian YouTube
- penyimpanan konfigurasi lokal atau penyimpanan lain sesuai implementasi masing-masing

Conot secara umum dirancang agar tidak bergantung pada YouTube Data API v3, tetapi fork atau turunan proyek dapat mengubah perilaku tersebut.

Karena proyek ini dapat bersifat open source:

- source code dapat dimodifikasi oleh pihak lain

- setiap operator instance dapat menambah, menghapus, atau mengubah perilaku bot
- maintainer asli tidak selalu mengendalikan setiap deployment yang beredar
